# ⚙️ .dotfiles

Personal configuration files for a high-performance **Linux** development environment. Optimized for **Neovim** power-use, efficient navigation, and full-stack development.

## 🖥️ Core Environment

* **OS:** Linux Enthusiast
* **Editor:** **Neovim** (Configured for Python, React, and C++)
* **Multiplexer:** **Tmux** for session persistence
* **Shell Tools:** `zoxide`, `fzf`, `eza`, `fd`, and `ripgrep`

## 🛠️ Key Workflows

* **Modern CLI:** Using `zoxide` for smarter navigation and `fzf` for fuzzy finding across the system.
* **Clean Code:** Configured LSP and formatting for **FastAPI (Python)** and **React (TypeScript)**.
* **Performance:** High-speed searching using `ripgrep (rg)`.

## 🚀 Installation
>
> **Warning:** These configurations are tailored to my specific workflow. Review before applying.

```bash
# Clone the repository
git clone [https://github.com/Xander993/dotfiles.git](https://github.com/Xander993/dotfiles.git) ~/dotfiles
cd ~/dotfiles

# I recommend manually symlinking or using a tool like GNU Stow
ln -s ~/dotfiles/.tmux.conf ~/.tmux.conf
ln -s ~/dotfiles/nvim ~/.config/nvim
```
